Utilize Your Power Automate Licenses

Flow Utilizer is a tool that helps you to utilize your Power Automate licenses by estimating and tracking API calls.

Handling the complexity of loop iterations and conditional branching, spread across mulitple flows ans solutions can be challenging, Flow Utilizer simplifes this for you. Key features include:

  • Manual estimations
  • Import solution estimations
  • Save multiple solutions
  • License recommendation
  • Daily tracking
  • Export and import data in YAML formt
  • Visulaised data in charts
  • Mobile and Desktop optimized

More information on limits can be found in Microsofts docs:https://learn.microsoft.com/en-us/power-platform/admin/api-request-limits-allocations
